# Task 2: Data Visualization and Storytelling

**Internship**: DataX Labs – Data Analyst Internship
**Tool**: Power BI Desktop
**Dataset**: Sample Superstore (Kaggle)

## Dashboard Includes
- KPI Cards: Total Sales & Total Profit
- Bar Chart: Sales & Profit by Category
- Line Chart: Sales Trend Over Time
- Map: Sales by State
- Treemap: Profit by Sub-Category (highlights loss-making items)

## Key Insight
Furniture shows high sales but low profit margins compared to Technology. 
Certain sub-categories show negative profit, indicating a need to review discounting strategy.
